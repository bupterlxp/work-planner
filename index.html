        快速添加语法

        在顶部输入框里，一行写完所有信息：

        • #项目名 — 归入项目（不存在会自动创建）
        • !高 / !中 / !低（或 !h !m !l）— 优先级
        • @今天 @明天 @后天 @周一…@周日 或 @2026-07-01 — 截止日期

        例：完成季度汇报 #规划 !高 @周五

        拖拽

        • 看板：拖动卡片在「待办 / 进行中 / 已完成」之间移动，或调整顺序。
        • 本周：把卡片拖到某一天即设为该天截止；拖到「无日期」清除日期。

        数据保存在本浏览器本地，定期用「导出备份」以防丢失。

        🗂️ 长期项目 & 短期任务

        • 长期项目是持续推进的大目标，带目标、状态、截止日和里程碑；短期任务归属于某个项目（编辑任务时选「项目」即可）。
        • 在「🗂️ 项目」视图里，每个长期项目一张卡，自动汇总旗下任务的进度与里程碑完成度；点卡片可编辑项目、管理里程碑与任务。各项目锻炼了你哪些能力，统一在「🧬 能力」面板的「📊 项目能力」里查看。

        🧬 能力面板 & 🤖 AI

        • 能力面板：每完成一个工作项，系统会分析它锻炼了哪些能力并加分、升级，长期沉淀为你的能力雷达图。
        • AI 总结：点右上角 🤖 一键生成今日/本周工作进展总结。默认离线本地引擎；在「AI 设置」里填入 OpenAI API Key 可切换到 OpenAI，更智能。

        ⚙️ AI 设置

        本地引擎用关键词规则分析，无需联网、不花钱；OpenAI 引擎用官方 SDK 调用你自己的 API，对总结与能力归因更精准。
        ⚠️ 密钥仅保存在你的浏览器本地（localStorage），并由你的浏览器直接请求 OpenAI。请勿在公用电脑上使用。
        用官方 OpenAI SDK 协议，可指向任何兼容 OpenAI的服务（DeepSeek / Moonshot / 通义 / OpenRouter / Ollama 等）。留空即用官方接口。注意：该地址需允许浏览器跨域（CORS）。
        从建议中选择，或手动输入你账号下可用的任意模型 ID。
